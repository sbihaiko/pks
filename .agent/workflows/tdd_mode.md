---
description: Usar TDD para desenvolver a funcionalidade.
---
1. Use @agents especializados se entender necessário.
2. Escreva um teste para a funcionalidade.
3. Execute o teste e verifique se ele falha.
4. Explique qual foi o erro e proponha uma correção.
5. Implemente a funcionalidade.
6. Execute o teste e verifique se ele passa.
7. Refatore o código se necessário.
7. repita os passos 1-6 até que a funcionalidade esteja implementada corretamente.
